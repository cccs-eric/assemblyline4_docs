# Alert
> The Alert model encapsulates all the relevant information generated when Assemblyline identifies a potential threat within a submitted file. It includes metadata like the alert's unique identifier, classification level, and detailed results of the analysis. This model serves as a comprehensive record of the threat detection process and its outcome.
